Content Aware Image Resizing

A discord bot that uses Computer Vision to resize an image by removing only the least noticeable pixels.

Uses Discord.js and Node.js to build a Canvas context from an image sent by the user on Discord. This information is then passed to a C++ program on the server that uses Dynamic Programming to efficiently detect and selectively remove least noticeable pixels. The re-sized image is then provided to the user by the bot through a message attachment on discord.

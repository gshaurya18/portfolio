Content Aware Image Resizing

A discord bot that uses Computer Vision to resize an image by removing only the least noticeable pixels.

Uses Discord.js and Node.js to build a Canvas context from an image sent by the user on Discord. This information is then passed to a C++ program on the server that uses Dynamic Programming to efficiently detect and selectively remove least noticeable pixels. The re-sized image is then provided to the user by the bot through a message attachment on discord.

Map building from LiDAR data

Uses a Kalman Filter Algorithm to process and transform point cloud data collected from a thirty-two beam LiDAR on a robot using Julia.

Parsed robot pose and point cloud data to calculate Linear Affine Transformations needed to bring data into the World Frame in real time. Fused Data collected collected over multiple time periods with the robot's motion to gain full scene understanding and build a map. The gif here shows the map built over 5 seconds as the robot walked towards a staircase.

Dog Breed Recognition

Trained a Convoluted Neural Network (CNN) to identify 8 species of dogs: samoyed, poodle, saint bernard, dalmatian, chihuahua, husky, and yorkshire terrier.

Preprocessed the images to make the underlying distribution of each image channel have mean and variance 0 and 1 respectively. Used Python PyTorch and TensorFlow to train a 6 layer CNN on about 9000 images of dogs labeled with their breeds. Correctly classified about 72% of test images, taking about 3 seconds to classify each individual image.
